# AWS Resource Listing Script

## Overview
This script lists AWS resources (EC2 instances, S3 buckets, DynamoDB tables) 
for a given service and region using boto3.

## Requirements
- Python 3.x
- boto3 (`pip install boto3`)
- AWS CLI configured with credentials (`aws configure`)

## Usage
```bash
python3 list_resources.py <service> <region>

ec2 â†’ Lists all EC2 instance IDs in the region.

# List DynamoDB tables in eu-west-1
