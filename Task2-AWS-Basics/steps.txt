1. Create EC2 Instance (t2.micro, Amazon Linux 2)
Option A: Using AWS Management Console

Log in to AWS Console
.

Navigate to EC2 → Instances → Launch Instance.

Enter a name (e.g., DevOps-Instance).

Choose Amazon Linux 2 AMI (64-bit x86) (latest version).

Choose Instance type: t2.micro (Free Tier eligible).

Configure Key pair (for SSH):

Create a new key pair (devops-key.pem) and download it.

Configure Security Group:

Allow SSH (port 22) from your IP.

Allow HTTP (port 80) from anywhere (so you can access the web page).

Leave storage defaults (8GB gp2).

Click Launch Instance.

Option B: Using AWS CLI

Make sure AWS CLI is configured:

aws configure


Find the latest Amazon Linux 2 AMI ID:

aws ec2 describe-images \
  --owners amazon \
  --filters "Name=name,Values=amzn2-ami-hvm-*-x86_64-gp2" "Name=state,Values=available" \
  --query "Images[*].[ImageId,CreationDate]" \
  --output text | sort -k2 -r | head -n1


Suppose it returns ami-0abcdef1234567890.

Create a key pair:

aws ec2 create-key-pair --key-name devops-key --query "KeyMaterial" --output text > devops-key.pem
chmod 400 devops-key.pem


Create a security group:

aws ec2 create-security-group --group-name devops-sg --description "DevOps Security Group"


Allow SSH & HTTP:

aws ec2 authorize-security-group-ingress --group-name devops-sg --protocol tcp --port 22 --cidr 0.0.0.0/0
aws ec2 authorize-security-group-ingress --group-name devops-sg --protocol tcp --port 80 --cidr 0.0.0.0/0


Launch instance:

aws ec2 run-instances \
  --image-id ami-0abcdef1234567890 \
  --count 1 \
  --instance-type t2.micro \
  --key-name devops-key \
  --security-groups devops-sg


Get the public IP:

aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=running" \
  --query "Reservations[*].Instances[*].PublicIpAddress" \
  --output text

2. SSH into EC2 Instance

Use the key pair:

ssh -i devops-key.pem ec2-user@<EC2_PUBLIC_IP>

3. Install and Configure Web Server
Option A: Apache
sudo yum update -y
sudo yum install -y httpd
sudo systemctl start httpd
sudo systemctl enable httpd
echo "Hello, DevOps!" | sudo tee /var/www/html/index.html

Option B: Nginx
sudo amazon-linux-extras enable nginx1
sudo yum install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
echo "Hello, DevOps!" | sudo tee /usr/share/nginx/html/index.html

4. Verify

Open a browser:

http://<EC2_PUBLIC_IP>/


You should see:

Hello, DevOps!
